# Google Apps Script Template (Legacy)

Folder ini berisi template lama yang masih disimpan sebagai referensi historis.

Untuk setup baru, gunakan template aktif berikut:

- `docs/spreadsheet-template.xlsx`
- `docs/spreadsheet-template/apps-script-example.gs`
- `docs/spreadsheet-template/*.csv`

Template aktif memakai tab:

1. `service_categories`
2. `services`
3. `promos`
4. `packages`
5. `testimonials`
6. `featured_treatments`
7. `about_values`
8. `branches`
9. `help_topics`
10. `site_content`

Jangan mencampur format lama di folder ini dengan template aktif, karena format lama memakai nama sheet berbeda seperti:

- `landing_hero`
- `landing_consultation`
- `landing_services`
- `landing_promos`
- `landing_testimonials`
- `service_list`
- `about_page`
- `help_page`

Jika sedang membuat spreadsheet baru, ikuti `docs/spreadsheet-template.md` atau `docs/spreadsheet-template/README.md`.
